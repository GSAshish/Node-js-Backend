math.mjs

export default const add=(a,b)=>a+b;

third way
multiple
math.mjs
```javascript
const add=(a,b)=>a+b;
const sub=(a,b)a-b;

```


`import math from "./math.mjs"`
or
`import {add,sub} from "./math.mjs"`

4th way
```javascript
export const add=(a,b)=>a+b;
export const sub=(a,b)a-b;

```


`import * as math from "./math.mjs"`
or
`import {add,sub} from "./math.mjs"`
